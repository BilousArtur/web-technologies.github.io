# Lab 1: Introduction to JavaScript

## Objective
Learn to work with the `<script>` tag, connect JavaScript to HTML, and set up interaction between HTML and scripts.

## Completed Tasks
- [x] Familiarization with the `<script>` tag
- [x] Connecting an external JavaScript file
- [x] Using DOM access methods (`getElementById`, `querySelector`)
- [x] Using HTML events (`onclick`)
- [x] Displaying "Hello world!" message

## Technologies Used
- HTML5
- CSS3
- JavaScript

## How to Run
1. Open `index.html` in any modern browser
2. Click the "Show Hello World" button
3. Or open the console (F12) to see results
