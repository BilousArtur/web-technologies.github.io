## 🚀 How to Run Any Lab
1. Navigate to the lab folder (e.g., `/labs/lab1`)
2. Open the `index.html` file in any modern browser
3. Open the browser console (F12) to see detailed output
4. Interact with buttons or elements as described in each lab
